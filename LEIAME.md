# archrice: dotfiles

### Sumário

Leia em [English](./README.md)

Minha configuração do Arch Linux. Inclui um número de parâmetros úteis para
alguns programas de linha de comando.

Configurações estão disponíveis para os programas abaixo:
* bash
* vim
* alacritty
* bat
* fontconfig
* gtk2
* gtk3
* i3
* lf
* mpd
* mpv
* ncmpcpp
* newsboat
* picom
* pipewire
* rofi
* xinit
* zathura
* dunst
